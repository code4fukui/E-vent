# E-vent

E-ventは、コミュニティイベントの企画・参加を支援するウェブプラットフォームです。

## 機能

- 日時や場所、説明などの詳細を含めたイベントの作成・管理
- イベント参加の申し込みとコメント投稿
- 新規イベントや締め切りの自動ツイート告知
- イベント主催者のための承認プロセス

## 必要条件

- Denoランタイム
- CloudinaryおよびTwitter APIの認証情報

## 使い方

Denoがインストールされていることを確認してください: https://deno.land/manual/getting_started/installation

その後、プロジェクトを起動します:

```
deno task start
```

これによりプロジェクトディレクトリが監視され、必要に応じて再起動されます。
