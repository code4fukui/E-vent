# E-vent

イベントの開催・参加者募集・感想の投稿ができるウェブアプリケーションです。

## デモ
[E-vent](https://e-vent.deno.dev/)

## 機能
- イベントの作成・承認
- イベントへの参加申請
- イベントに対するコメントの投稿
- 開催予定のイベント一覧の表示
- 過去のイベントの閲覧

## 必要環境
- Deno

## 使い方
Denoをインストールした上で、以下のコマンドを実行します:

```
deno task start
```

これによってアプリケーションが起動します。
